## Run Locally

Clone the project

```bash
  git clone https://github.com/kerBiy/web-portfolio.git
```

Go to the project directory

```bash
  cd web-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
